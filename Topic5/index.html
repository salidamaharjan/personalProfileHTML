CST-120 Web Development Syllabus

Course Description

This course introduces the fundamental concepts and syntax of the web development languages including HTML, CSS, and JavaScript. The course focuses on foundation required to build complex dynamic web applications. Prerequisite: none.

Weeks 1-2

Topic 1 HTML

In this topic, students are introduced to the HTML markup language where they learn basic structure, basic elements of an HTML document, and how to work with text and basic controls. The students finish this topic by learning the developer tools available to them, as well as how web servers work to help support development of HTML-based applications.

  1. Develop webpages with basic elements and structure that include, but are not limited to, document, head, and body tags.
  2. Develop webpages with text that includes, but is not limited to, headings, paragraphs, formatting, fonts, and color tags.
  3. Develop webpages with components that include, but are not limited to, links, images, and lists tags.
  4. Investigate how a web server functions. Identify differences between localhost, file URL and domain names in the URL.
  5. Investigate the developer's tools available as a webpage developer.
